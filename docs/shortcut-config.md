# Shortcut Configuration

Default shortcuts:

```json
{
  "toggleChat": "Ctrl+Alt+W",
  "hidePet": "Ctrl+Alt+H",
  "centerPet": "Ctrl+Alt+B",
  "quickAsk": "Ctrl+Alt+Space"
}
```

Quick commands can also have shortcuts:

```json
{
  "id": "daily-plan",
  "name": "Daily Plan",
  "shortcut": "Ctrl+Alt+1",
  "prompt": "Help me organize today's work plan by priority."
}
```

Desktop global shortcuts are registered by Tauri. Browser preview shortcuts are handled inside the active browser window.
